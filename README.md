## 项目说明

当前版本: 0.0.1

该项目是一个 nodejs + koa2 + mysql 的后端项目模板

该项目模块化规范采用 ESModule

项目中配置了 router , mysql , JWT , log



## 下载

**使用git**

```git
git clone https://github.com/rgbjs/project-koa2.git
```



## 使用

**开发模式**

```cmd
pnpm run dev
# or
yarn run dev
# or
npm run dev
```



**生产模式**

```cmd
pnpm run start
# or
yarn run start
# or
npm run start
```
