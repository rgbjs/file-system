## 项目说明

当前版本: 0.0.3

该项目是一个 nodejs + koa2 + mysql 的后端项目模板

该项目模块化规范采用 ESModule

项目中配置了 router , mysql , JWT , log



## 下载

**使用git**

```git
git clone https://github.com/rgbjs/project-koa2.git
```



## 使用

**安装依赖**

*pnpm*

```cmd
pnpm i
```

*yarn*

```cmd
yarn install
```

*npm*

```cmd
npm i
```





**开发模式**

*pnpm*

```cmd
pnpm run dev
```

*yarn*

```cmd
yarn run dev
```

*npm*

```cmd
npm run dev
```



**生产模式**

*pnpm*

```cmd
pnpm run start
```

*yarn*

```cmd
yarn run start
```

*npm*

```cmd
npm run start
```
